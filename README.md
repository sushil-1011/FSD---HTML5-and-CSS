# HTML5 and CSS – Basic Concepts and Styling

## Project Overview

This repository contains simple HTML5 and CSS practice files created as part of Full Stack Development learning.

The project demonstrates:

- HTML5 semantic elements
- Audio and video embedding
- Different types of styling in HTML
- Basic CSS usage

All files are developed to understand modern HTML5 features and styling concepts.


## Objectives

- Learn HTML5 semantic tags like article
- Understand how to embed audio and video
- Practice different types of CSS styling
- Improve webpage design using CSS
- Strengthen frontend development basics


## Technologies Used

- HTML5
- CSS3


## Project Structure

```
HTML5-CSS-Basics/
```

## File Description

### 1. article.html
Demonstrates HTML5 semantic elements such as:
- article
- section
- header
- footer

Shows how content can be structured properly using semantic tags.

### 2. audio-video.html
Demonstrates how to embed media in a webpage using:
- audio tag
- video tag
- controls attribute
- source tag

### 3. styles.html
Demonstrates different types of styling in HTML:
- Inline CSS
- Internal CSS
- External CSS

### 4. style.css
Contains external CSS styles used to design and format HTML elements.


## Types of CSS Used

1. Inline CSS  
   Styling applied directly inside an HTML tag using the style attribute.

2. Internal CSS  
   Styling written inside the style tag within the head section of the HTML file.

3. External CSS  
   Styling written in a separate .css file and linked using the link tag.


## How to Run

1. Clone the repository:

```
git clone <repository-link>
cd HTML5-CSS-Basics
```

2. Open any HTML file in a web browser.

No additional installation is required.
